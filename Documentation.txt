Documentation

* Download virtualenv
* Make project directory
* change to that directory
* run $virtualenv venv
This will make a folder called "venv" in your project directory that will contain everything you download for project environment.
* run $. venv/bin/activate to activate the virtual environment (to deactivate run $deactivate)
* install Flask in your venv environment
* pip install --upgrade google-api-python-client
* pip install Flask-Assets  ~~ to use js and bootstrap
